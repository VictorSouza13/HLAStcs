HLAStcs, an R package for in silico analysis of HLA–peptide interactions.

HLAStcs provides tools for handling peptide sequences, encoding amino acid features, and supporting computational workflows related to immunogenetics and antigen presentation studies.

Main features:
  Functions for peptide sequence manipulation and preprocessing
  Tools for encoding amino acid properties
  Utilities for preparing datasets for machine learning approaches
  Support for workflows related to HLA–peptide binding analysis
